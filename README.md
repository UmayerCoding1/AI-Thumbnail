# 🎨 AI Thumbnail Generator

**AI Thumbnail Generator** is a modern, AI-powered web application designed to help creators generate stunning, high-converting thumbnails for their videos in seconds. Leveraging advanced AI models and a sleek, intuitive interface, AI Thumbnail Generator streamlines the creative process from prompt to preview.

---

## ✨ Key Features

- **AI-Powered Generation**: Transform your ideas into professional thumbnails using state-of-the-art AI.
- **Style Selector**: Choose from a wide range of artistic styles (e.g., Cyberpunk, Minimalist, Cinematic) to match your brand.
- **Aspect Ratio Control**: Optimize your thumbnails for YouTube (16:9), Instagram (9:16), or standard squares (1:1).
- **Color Scheme Customization**: Fine-tune the mood with curated color palettes.
- **Live YouTube Preview**: Visualize your thumbnail in a realistic YouTube interface before downloading.
- **My Generations**: A dedicated gallery to manage, view, and download all your previously created thumbnails.
- **Smooth UX**: Built with Lenis smooth scrolling and fluid Framer Motion animations for a premium feel.

---

## 🚀 Tech Stack

- **Core**: [Next.js 16](https://nextjs.org/) (App Router), [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Utilities**: 
  - [Lenis](https://lenis.darkroom.engineering/) (Smooth Scrolling)
  - [React Fast Marquee](https://www.react-fast-marquee.com/)
  - [Tailwind Merge](https://github.com/dcastil/tailwind-merge) & [clsx](https://github.com/lukeed/clsx)

## 🛠️ Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm / yarn / pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone [repository-url]
   ```
2. Navigate to the client directory:
   ```bash
   cd client
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
